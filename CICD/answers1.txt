1. Why should you use a version control tool?

		We use a version control tool to store the back up of the source code of a working product with a specification of the version. If any improvement is made to the code and the code doesn't generate desired output, then the backup created can be reused to recreate the working product.

7. How is system testing different from unit testing and Integration testing?

9. What is meant by Regression testing?

		Regression testing is the process of testing the product after the increment of a few modules to the system, whether the system was able to handle the changes and it still works well.
